- **Workspace 資源**：所有 agent 共享同一個目錄（/home/tsukii0607/.openclaw/workspace），包括檔案、MEMORY.md、memory/*.md、日誌等。修改一處，全域生效。

- **系統配置**：Gateway、cron jobs、channel 設定（e.g. Telegram bot token）係共通，agent 間可以互傳訊息（用 sessions_send）。
- **記憶與上下文**：長期記憶（MEMORY.md）只喺 main session 載入，但 daily memory 檔案可跨 agent 參考。

獨立嘅係每個 agent/session 嘅對話歷史同狀態（用 sessions_spawn 創 sub-agent 時會 isolated）。如果要分開 bot，建議用唔同 agentId 配 sessionKey 管理，共通資源就自動同步。
